# 玄股 - 以天道观股道

基于中国古代玄学的 A 股分析静态网站。

## 功能特点

- **今日大势**：展示当日农历、天干地支、建除十二神、二十八星宿、九宫飞星等玄学信息，并给出大盘走势预测
- **板块轮动**：将 A 股板块按五行（金木水火土）分类，通过罗盘可视化展示各板块运势

## 玄学要素

本项目使用 `lunar-javascript` 库获取以下玄学信息：

- **十二建除**：建、除、满、平、定、执、破、危、成、收、开、闭
- **二十八星宿**：角、亢、氐、房、心、尾、箕等
- **九宫飞星**：一白贪狼至九紫右弼
- **神煞**：天德、月德、大耗、小耗等吉凶神煞
- **天干地支**：年月日时四柱

## 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion
- lunar-javascript

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
