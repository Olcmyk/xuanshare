# 玄股 - 以天道观股道

> 将中国古代玄学智慧与A股分析相结合的创意应用

## 项目简介

玄股是一个独特的股票分析工具，它将传统玄学体系（八字命理、五行学说、建除十二神、二十八星宿、九宫飞星等）与现代股市分析相融合，为投资者提供一个全新的观察视角。

**⚠️ 免责声明：本项目内容仅供娱乐参考，不构成任何投资建议。股市有风险，投资需谨慎。**

## 玄学与股市的对应体系

### 一、五行与板块映射

五行是整个系统的基础框架。按照行业的本质属性，将A股板块归类到五行之中：

| 五行 | 属性特征 | 对应板块 |
|------|---------|---------|
| **金** | 坚硬、收敛、肃杀 | 有色金属、贵金属、钢铁、银行、保险、证券、汽车、机械设备、国防军工 |
| **木** | 生长、条达、舒展 | 农林牧渔、家具、造纸、纺织服装、医药生物、教育、传媒、文化、出版 |
| **水** | 流动、润下、智慧 | 航运、港口、水务、饮料、酿酒、旅游、酒店、物流、贸易 |
| **火** | 炎上、光明、急速 | 电力、光伏、新能源、电子、半导体、通信、计算机、互联网、传媒娱乐 |
| **土** | 承载、稳固、厚重 | 房地产、建筑、建材、水泥、玻璃、陶瓷、基建、工程机械、环保 |

**五行相生相克与板块轮动**：
- 相生：木→火→土→金→水→木（前者利好后者）
- 相克：木克土、土克水、水克火、火克金、金克木（前者压制后者）

当某一五行当令（强旺）时，其对应板块往往表现活跃；同时会生助下一个五行的板块，压制所克五行的板块。

### 二、建除十二神与操作策略

建除十二神是古代择日学的核心，本项目将其映射为12种不同的市场操作建议：

| 建除 | 市场含义 | 宜 | 忌 |
|------|---------|-----|-----|
| **建** | 建仓良机 | 建仓、开户、申购新股、长线布局 | 清仓、止损 |
| **除** | 清除杂念 | 清理亏损股、调整持仓、止损出局 | 追高、加仓 |
| **满** | 获利了结 | 获利了结、分批减仓、落袋为安 | 追涨、满仓操作 |
| **平** | 平稳持有 | 持股观望、平衡配置、稳健操作 | 激进操作、频繁交易 |
| **定** | 坚定持有 | 坚定持股、定投加仓、长期持有 | 恐慌抛售、频繁换股 |
| **执** | 执行计划 | 执行既定策略、按计划操作、纪律交易 | 临时改变策略、情绪化操作 |
| **破** | 谨慎观望 | 观望、空仓休息、研究学习 | 重仓操作、追涨杀跌、借钱炒股 |
| **危** | 规避风险 | 降低仓位、设置止损、防守为主 | 满仓操作、追高、杠杆交易 |
| **成** | 成交活跃 | 短线交易、波段操作、把握机会 | 犹豫不决、错失良机 |
| **收** | 收获成果 | 收割利润、兑现收益、总结复盘 | 贪心不足、恋战不走 |
| **开** | 开拓进取 | 开新仓、尝试新板块、扩大投资 | 保守观望、错失机会 |
| **闭** | 闭门休息 | 空仓休息、闭关学习、静待时机 | 频繁操作、盲目跟风 |

### 三、二十八星宿与市场信号

二十八星宿各有吉凶属性，对应不同的市场信号。例如：
- **角宿（吉）**：龙头股活跃、科技板块有机会
- **亢宿（凶）**：高位股有回调风险、谨慎追高
- **氐宿（吉）**：底部股有反弹机会

每个星宿还对应特定的板块提示，为投资者提供参考方向。

### 四、九宫飞星与大盘走势

九宫飞星（1-9）代表不同的市场能量：

| 飞星 | 五行 | 市场信号 |
|------|------|---------|
| 1白贪狼 | 水 | 水务、物流板块活跃 |
| 2黑巨门 | 土 | 地产板块承压 |
| 3碧禄存 | 木 | 农业、医药板块有机会 |
| 4绿文曲 | 木 | 教育、文化板块活跃 |
| 5黄廉贞 | 土 | 五黄煞临，大盘承压 |
| 6白武曲 | 金 | 金融、军工板块有机会 |
| 7赤破军 | 金 | 市场波动加大 |
| 8白左辅 | 土 | 财星高照，利于投资理财 |
| 9紫右弼 | 火 | 科技、新能源板块活跃 |

### 五、神煞系统与风险提示

神煞分为吉神和凶神，共40余种：

**吉神示例**：
- 天德、月德：贵人相助，市场有支撑
- 天财、天喜：财运亨通，市场情绪高涨
- 驿马：物流交通板块活跃，资金流动快

**凶神示例**：
- 大耗、小耗：资金外流，消耗多头动能
- 五黄：全面回调风险
- 岁破、月破：年线/月线级别风险

### 六、个股八字分析

将股票的上市日期视为其"生辰"，进行八字命理分析：

1. **四柱推算**：根据上市日期计算年柱、月柱、日柱、时柱
2. **日主五行**：确定股票的核心五行属性
3. **五行强弱**：分析股票八字中各五行的旺衰
4. **今日兼容度**：计算当日天干地支与股票八字的相合程度
   - 今日天干生股票日主 → 贵人相助，利好
   - 股票克今日天干 → 财星动，利好
   - 今日克股票 → 官星临，压力
   - 股票生今日天干 → 泄气，不利

### 七、个人命理与投资

根据用户的生辰八字，分析其投资特质：

1. **用神判断**：根据八字强弱确定用神（对自己有利的五行）
2. **有缘板块**：用神对应的五行板块
3. **有缘股票**：与用神、财星相合的股票
4. **幸运元素**：幸运数字、颜色、日期等

## 核心算法简述

### 市场预测综合评分

```
综合评分 = 建除影响 + 星宿影响 + 九宫飞星影响 + 天神吉凶 + 神煞影响

建除影响：建/开/成/满 → +2分，破/危/闭 → -2分
星宿影响：吉星 → +1分，凶星 → -1分
九宫飞星：1/6/8 → +1分，2/5/7 → -1分
神煞影响：每个吉神 +0.5分，每个凶神 -0.5分

最终判断：
- 总分 > 1：看涨
- 总分 < -1：看跌
- 其他：震荡
```

### 五行强弱计算

```
各柱权重：
- 月支（月令）：权重最高，主导当月气候
- 日干：权重次高，代表核心属性
- 其他干支：按位置分配权重

最终归一化到 1-5 的强度等级
```

## 技术栈

- **框架**：React 19 + TypeScript
